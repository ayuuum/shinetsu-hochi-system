# 信越報知システム 利用者マニュアル

発行日: 2026年5月1日  
対象: 作業者、管理者、人事担当者  
用途: 社内配布、初回説明、操作確認、印刷・PDF保存

---

## 1. このシステムでできること

信越報知システムは、社員情報、資格期限、施工実績、車両・備品、健康診断、アルコールチェックを一元管理するための業務システムです。

日常的には、次の目的で使います。

- 作業者が、今日必要な確認とアルコールチェックを行う
- 管理者・人事担当者が、社員台帳や資格期限を管理する
- 期限切れ、期限間近、未記録、不適正記録を画面上で確認する
- 社員情報やアルコールチェック記録をCSV・Excelで出力する
- 既存Excelの社員情報をCSVに変換して一括登録する

メールによる業務アラート通知は使いません。期限や未記録の確認は、ダッシュボードと各一覧画面で行います。

---

## 2. 利用者の種類

| 利用者 | 主な入口 | できること |
|---|---|---|
| 作業者 | 今日の作業 | 自分のアルコールチェック、自分の資格期限、自分の基本情報を確認 |
| 人事 | ダッシュボード | 社員台帳、資格、健康診断などの管理 |
| 管理者 | ダッシュボード | 全機能の管理、ユーザー招待、権限変更、社員紐づけ |

ログイン画面は全員共通です。ログイン後、権限に応じて表示されるホーム画面とメニューが変わります。

---

## 3. ログインとパスワード

### 3.1 ログインする

1. ブラウザでログイン画面を開きます。
2. メールアドレスとパスワードを入力します。
3. `ログイン` を押します。
4. 権限に応じて、作業者は `今日の作業`、管理者・人事は `ダッシュボード` に移動します。

### 3.2 パスワードを忘れた場合

1. ログイン画面の `パスワードを忘れた場合` を押します。
2. 登録済みメールアドレスを入力します。
3. 届いたメール内のリンクを開きます。
4. 新しいパスワードを入力して保存します。
5. ログイン画面から新しいパスワードでログインします。

### 3.3 認証メールについて

業務アラートのメール通知は使いませんが、次の認証メールはログイン運用に必要です。

- ユーザー招待メール
- パスワード再設定メール

メールが届かない場合は、迷惑メールフォルダ、メールアドレスの入力間違い、リンクの有効期限を確認してください。

---

## 4. 作業者向け操作

作業者は、毎日の入力と自分の情報確認に必要な画面だけを使います。

### 4.1 今日の作業を確認する

`今日の作業` では、当日確認すべき内容をまとめて確認できます。

確認できる内容:

- 本日のアルコールチェック状況
- 自分の資格期限
- 自分のプロフィールへの導線

### 4.2 アルコールチェックを記録する

1. 下部メニューまたは画面内の `アルコールチェック` を開きます。
2. `体調・アルコール記録` を押します。
3. 出勤時または退勤時の記録を入力します。
4. アルコール検知の有無、体調、確認者などを入力します。
5. 内容を確認して登録します。

注意:

- 不適正の記録がある場合は、画面表示に従って安全運転管理者へ報告してください。
- 作業者は、通常は自分以外の社員の記録を操作できません。

### 4.3 自分の資格を確認する

1. `今日の作業` から `マイプロフィールを見る` を押します。
2. `保有資格` タブを開きます。
3. 期限切れや期限間近の資格がないか確認します。

資格期限の目安:

| 表示 | 意味 |
|---|---|
| 期限切れ | すでに有効期限を過ぎています |
| 14日以内 | 早急な対応が必要です |
| 30日以内 | 更新準備が必要です |
| 60日以内 | 今後の予定として確認してください |

### 4.4 自分の基本情報を確認する

1. `マイプロフィールを見る` を押します。
2. `基本情報` タブを開きます。
3. 住所、連絡先、入社日などを確認します。

内容に誤りがある場合は、管理者または人事担当者へ連絡してください。

## 5. 管理者・人事向け操作

管理者・人事担当者は、会社全体の状態確認と台帳管理を行います。

### 5.1 ダッシュボード

`ダッシュボード` では、会社全体の状況を確認できます。

確認できる内容:

- 最優先タスク
- 資格期限のアラート
- アルコールチェックの未記録・不適正
- 車検・保険などの期限
- 今月の予定・期限

日常的には、まずダッシュボードを確認し、必要な画面へ移動します。

### 5.2 社員台帳

`社員台帳` では、社員情報の確認・登録・編集を行います。

主な操作:

- 社員を検索する
- 支店や資格で絞り込む
- 社員を新規登録する
- 社員情報を編集する
- 顔写真、家族、保険、IT利用情報を確認する
- 施工実績、健康診断、資格履歴を確認する
- CSV出力を行う

社員詳細の主なタブ:

| タブ | 内容 |
|---|---|
| 基本情報 | 住所、連絡先、雇用情報 |
| 保有資格 | 社員が持つ資格と期限 |
| 施工実績 | 担当した施工実績 |
| 健康診断 | 健康診断の履歴 |
| 家族 | 家族・緊急連絡先 |
| IT・ライセンス | ITアカウントや利用サービス |
| 保険情報 | 生命保険・損害保険 |

### 5.3 資格・講習管理

`資格・講習管理` では、社員の資格期限と講習履歴を管理します。

主な操作:

- 資格を検索する
- 期限状態で絞り込む
- 資格情報を追加・編集する
- 講習履歴を登録する
- 証書画像を確認する
- 複数資格を一括更新する

### 5.4 資格マスタ

`資格マスタ` では、資格の種類を管理します。

主な操作:

- 資格名を追加する
- カテゴリを設定する
- 不要な資格マスタを削除する

資格マスタは、社員に資格を登録するときの選択肢になります。変更すると、今後の入力画面にも影響します。

### 5.5 工事経歴

`工事経歴` では、施工実績を管理します。

主な操作:

- 工事名、場所、担当者で検索する
- 工事種別で絞り込む
- 施工記録を追加・編集する
- Excel出力を行う
- 社員詳細から担当履歴を確認する

施工実績書の正式テンプレート対応は、既存様式の確認後に別途対応します。

### 5.6 車両・備品

`車両・備品` では、社用車と備品を管理します。

車両で確認できる内容:

- 車両名
- ナンバー
- 使用者
- 車検期限
- 自賠責保険期限
- 任意保険期限
- 修理履歴
- 事故履歴
- タイヤ履歴

備品で確認できる内容:

- 管理番号
- 備品名
- カテゴリ
- 購入日
- 保管場所
- 備考

### 5.7 健康診断管理

`健康診断管理` では、社員の健康診断履歴を管理します。

主な操作:

- 社員名で検索する
- 健診種別で絞り込む
- 結果で絞り込む
- 健康診断記録を追加・編集する

### 5.8 アルコールチェック管理

`アルコールチェック` では、日々のアルコールチェック記録を確認します。

主な操作:

- 日付で絞り込む
- 社員で絞り込む
- 拠点で絞り込む
- 不適正記録だけを確認する
- 月次記録率を確認する
- CSVや月次出力を行う

確認のポイント:

- 未記録者がいないか確認します。
- 不適正記録がある場合は、状況を確認します。
- 月次記録率で記録漏れの傾向を確認します。

### 5.9 年間スケジュール・目標

`年間スケジュール・目標` では、年度ごとの予定や目標を管理します。

主な操作:

- 年度を切り替える
- スケジュールを追加する
- スケジュールを編集する
- スケジュールを削除する

## 6. データインポート

`データインポート` では、CSVを使って社員データを一括登録します。

### 6.1 基本の流れ

1. 既存Excelを開き、社員情報を整理します。
2. システムのデータインポート画面からテンプレートCSVをダウンロードします。
3. テンプレートの列に合わせてデータを貼り付けます。
4. ExcelからCSV形式で保存します。
5. CSVをアップロードし、事前チェック結果を確認します。
6. エラーがある場合は、エラーCSVを出力して修正します。
7. 問題がなければ取り込みを実行します。
8. 取り込み履歴を確認します。

Excelファイルは直接取り込まず、必ずCSV形式に保存してからアップロードしてください。

### 6.2 CSVの必須項目

必須列:

- 社員番号
- 氏名
- フリガナ
- 生年月日
- 入社日
- 拠点

日付形式:

- `YYYY-MM-DD`
- `YYYY/M/D`

1回に取り込める上限は1,500行です。

---

## 7. ユーザー管理

`ユーザー管理` では、システムにログインできるユーザーを管理します。

主な操作:

- ユーザーを招待する
- 権限を変更する
- ユーザーを削除する
- 作業者アカウントと社員情報を紐づける

### 7.1 ユーザーを招待する

1. 管理者が `ユーザー管理` を開きます。
2. `招待する` を押します。
3. メールアドレスと権限を選択します。
4. 作業者の場合は、必要に応じて社員情報を紐づけます。
5. 招待を送信します。
6. 利用者に招待メールが届きます。
7. 利用者はメール内のリンクからパスワードを設定します。

### 7.2 権限の種類

| 権限 | 説明 |
|---|---|
| 管理者 | すべての管理機能を利用できます |
| 人事 | 社員・資格・記録管理などを利用できます |
| 技術者 | 自分の作業ページと自分に関係する情報を利用できます |

作業者が自分の情報を見られない場合は、ユーザー管理で社員情報との紐づけを確認してください。

---

## 8. 検索・絞り込み・出力

多くの一覧画面では、検索と絞り込みが使えます。

基本操作:

1. 検索欄に名前、番号、キーワードを入力します。
2. 必要に応じてカテゴリや状態を選びます。
3. 条件を解除したい場合は、クリア操作を行います。

主な出力:

- 社員台帳のCSV
- 施工実績のExcel
- アルコールチェックのCSV
- アルコールチェックの月次出力
- 社員の資格証書シート
- 社員の施工実績書

出力ボタンが表示されない場合は、権限が不足している可能性があります。

---

## 9. よくある困りごと

| 困りごと | 確認すること |
|---|---|
| ログインできない | メールアドレス、パスワード、パスワード再設定を確認 |
| メールが届かない | 迷惑メール、メールアドレス、リンク期限を確認 |
| 見たいメニューが表示されない | 権限が不足している可能性があります |
| 自分の情報が表示されない | 作業者アカウントと社員情報の紐づけを確認 |
| 登録・編集ができない | 権限が不足している可能性があります |
| 期限や件数が想定と違う | 検索条件や絞り込み条件を解除して再確認 |
| CSVインポートで失敗する | 必須列、日付形式、文字コード、1,500行制限を確認 |

---

## 10. 運用時の注意

- 登録前に入力内容を確認してください。
- 削除操作は慎重に行ってください。
- 期限切れや不適正記録は、確認後に担当者へ連絡してください。
- メール通知は運用対象外です。期限切れや期限間近の確認はダッシュボードと一覧画面で行ってください。
- 招待メール・パスワード再設定メールは、ログインのための認証メールとして使用します。
- CSVインポートは、事前チェック結果を確認してから実行してください。
- 権限変更は、利用者が見られる画面に影響します。

## 11. 初回利用時チェックリスト

管理者:

- 管理者アカウントでログインできる
- ダッシュボードが表示される
- 社員台帳を開ける
- ユーザー管理を開ける
- 作業者アカウントと社員情報を紐づけられる
- 招待メールを送信できる

作業者:

- 招待メールからパスワードを設定できる
- ログイン後に `今日の作業` が表示される
- アルコールチェックを記録できる
- `マイプロフィール` で自分の情報を確認できる

CSV移行:

- テンプレートCSVをダウンロードできる
- 既存ExcelをCSV形式で保存できる
- 事前チェック結果を確認できる
- エラーCSVを出力して修正できる
- 取り込み履歴を確認できる
