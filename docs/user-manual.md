# 信越報知システム 操作マニュアル

このマニュアルは、信越報知システムを利用する方向けの操作説明です。

ログイン画面は全員共通です。ログイン後、権限に応じて表示される入口とメニューが変わります。

- 作業者: 今日の作業ページに移動します。
- 管理者・人事: 管理ダッシュボードに移動します。

配布・印刷用のマニュアルは、アプリの `/manual` から開けます。ブラウザの印刷機能でPDF保存できます。

## 1. ログイン

1. ブラウザでシステムのログイン画面を開きます。
2. メールアドレスとパスワードを入力します。
3. `ログイン` を押します。

ログイン後の移動先:

- 作業者は `今日の作業` に移動します。
- 管理者・人事は `ダッシュボード` に移動します。

パスワードを忘れた場合:

1. ログイン画面の `パスワードを忘れた場合` を押します。
2. 登録済みメールアドレスを入力します。
3. 届いたメールのリンクから新しいパスワードを設定します。

## 2. 作業者向け

作業者は、毎日の確認と自分の情報確認に必要な画面だけを使います。

### 今日の作業

`今日の作業` では、当日に確認すべき内容をまとめて確認できます。

確認できる内容:

- 本日のアルコールチェック状況
- 自分の資格期限
- 自分のプロフィールへの導線

### アルコールチェックを記録する

1. 下部メニューまたは画面内の `アルコールチェック` を開きます。
2. `体調・アルコール記録` を押します。
3. 出勤時または退勤時の記録を入力します。
4. アルコール検知の有無、体調、確認者などを入力します。
5. 内容を確認して登録します。

注意:

- 不適正の記録がある場合は、画面表示に従って安全運転管理者へ報告してください。
- 自分以外の社員の記録は、通常は表示・操作できません。

### 自分の資格を確認する

1. `今日の作業` から `マイプロフィールを見る` を押します。
2. `保有資格` タブを確認します。
3. 期限切れや期限間近の資格がないか確認します。

資格の見方:

- `期限切れ`: すでに有効期限を過ぎています。
- `14日以内`: 早急な対応が必要です。
- `30日以内`: 更新準備が必要です。
- `60日以内`: 今後の予定として確認してください。

### 自分の基本情報を確認する

1. `マイプロフィールを見る` を押します。
2. `基本情報` タブを開きます。
3. 住所、連絡先、入社日などを確認します。

内容に誤りがある場合は、管理者または人事担当者へ連絡してください。

## 3. 管理者・人事向け

管理者・人事は、社員情報、資格、車両、健康診断、アルコールチェックなどを管理します。

### ダッシュボード

`ダッシュボード` では、会社全体の状況を確認できます。

確認できる内容:

- 最優先タスク
- 資格期限のアラート
- アルコールチェックの未記録・不適正
- 車検・保険などの期限
- 今月の予定・期限

日常的には、まずダッシュボードを確認し、必要な画面へ移動します。

### 社員台帳

`社員台帳` では、社員情報の確認・登録・編集を行います。

主な操作:

- 社員を検索する
- 支店や資格で絞り込む
- 社員を新規登録する
- 社員情報を編集する
- 顔写真、家族、保険、IT利用情報を確認する
- 施工実績、健康診断、資格履歴を確認する
- CSV出力を行う

社員詳細の主なタブ:

- `保有資格`: 社員が持つ資格と期限を確認します。
- `基本情報`: 住所、連絡先、雇用情報を確認します。
- `施工実績`: 担当した施工実績を確認します。
- `健康診断`: 健康診断の履歴を確認します。
- `家族`: 家族・緊急連絡先を確認します。
- `IT・ライセンス`: ITアカウントや利用サービスを確認します。
- `保険情報`: 生命保険・損害保険の情報を確認します。

### 資格・講習管理

`資格・講習管理` では、社員の資格期限と講習履歴を管理します。

主な操作:

- 資格を検索する
- 期限状態で絞り込む
- 資格情報を追加・編集する
- 講習履歴を登録する
- 証書画像を確認する
- 複数資格を一括更新する

期限状態の目安:

- `期限切れ`: 更新が必要です。
- `14日以内`: 優先して確認します。
- `30日以内`: 更新予定を確認します。
- `60日以内`: 今後の予定として管理します。

### 資格マスタ

`資格マスタ` では、資格の種類を管理します。

主な操作:

- 資格名を追加する
- カテゴリを設定する
- 不要な資格マスタを削除する

注意:

- 資格マスタは、社員に資格を登録するときの選択肢になります。
- 変更すると、今後の入力画面にも影響します。

### 工事経歴

`工事経歴` では、施工実績を管理します。

主な操作:

- 工事名、場所、担当者で検索する
- 工事種別で絞り込む
- 施工記録を追加・編集する
- Excel出力を行う
- 社員詳細から担当履歴を確認する

### 車両・備品

`車両・備品` では、社用車と備品を管理します。

車両で確認できる内容:

- 車両名
- ナンバー
- 使用者
- 車検期限
- 自賠責保険期限
- 任意保険期限
- 修理履歴
- 事故履歴
- タイヤ履歴

備品で確認できる内容:

- 管理番号
- 備品名
- カテゴリ
- 購入日
- 保管場所
- 備考

### 健康診断管理

`健康診断管理` では、社員の健康診断履歴を管理します。

主な操作:

- 社員名で検索する
- 健診種別で絞り込む
- 結果で絞り込む
- 健康診断記録を追加・編集する

### アルコールチェック管理

`アルコールチェック` では、日々のアルコールチェック記録を確認します。

主な操作:

- 日付で絞り込む
- 社員で絞り込む
- 拠点で絞り込む
- 不適正記録だけを確認する
- 月次記録率を確認する
- CSVや月次出力を行う

確認のポイント:

- 未記録者がいないか確認します。
- 不適正記録がある場合は、状況を確認します。
- 月次記録率で記録漏れの傾向を確認します。

### 年間スケジュール・目標

`年間スケジュール・目標` では、年度ごとの予定や目標を管理します。

主な操作:

- 年度を切り替える
- スケジュールを追加する
- スケジュールを編集する
- スケジュールを削除する

### データインポート

`データインポート` では、CSVを使って社員データを一括登録します。

基本の流れ:

1. CSVファイルを準備します。
2. インポート画面でファイルを選択します。
3. 事前チェック結果を確認します。
4. 問題がなければ取り込みを実行します。
5. 取り込み履歴を確認します。

注意:

- 取り込み前に、事前チェックのエラー内容を確認してください。
- Excelファイルは直接取り込まず、テンプレートCSVに合わせてCSV形式で保存してからアップロードしてください。
- 必須列は、社員番号、氏名、フリガナ、生年月日、入社日、拠点です。
- 日付形式は `YYYY-MM-DD` または `YYYY/M/D` にしてください。
- 1回に取り込める上限は1,500行です。
- 既存データに影響する可能性があるため、不明点がある場合は実行前に確認してください。

### ユーザー管理

`ユーザー管理` では、システムにログインできるユーザーを管理します。

主な操作:

- ユーザーを招待する
- 権限を変更する
- ユーザーを削除する
- 作業者アカウントと社員情報を紐づける

権限の種類:

- `管理者`: すべての管理機能を利用できます。
- `人事`: 社員・資格・記録管理などを利用できます。
- `技術者`: 自分の作業ページと自分に関係する情報を利用できます。

## 4. 検索と絞り込み

多くの一覧画面では、検索と絞り込みが使えます。

基本操作:

1. 検索欄に名前、番号、キーワードを入力します。
2. 必要に応じてカテゴリや状態を選びます。
3. 条件を解除したい場合は、クリア操作を行います。

画面上部の検索ボタンから、社員、資格、工事、車両、画面名を横断検索できます。

## 5. 出力

画面によって、CSV、Excel、印刷用ページの出力ができます。

主な出力:

- 社員台帳のCSV
- 施工実績のExcel
- アルコールチェックのCSV
- アルコールチェックの月次出力
- 社員の資格証書シート
- 社員の施工実績書

出力ボタンが表示されない場合は、権限が不足している可能性があります。

## 6. よくある困りごと

### ログインできない

- メールアドレスが正しいか確認してください。
- パスワードが正しいか確認してください。
- パスワードを忘れた場合は、ログイン画面から再設定してください。

### 見たいメニューが表示されない

権限によって表示されるメニューが異なります。

- 作業者には管理者向けメニューは表示されません。
- 必要なメニューが表示されない場合は、管理者に権限を確認してください。

### 自分の情報が表示されない

作業者アカウントに社員情報が紐づいていない可能性があります。

管理者に、ユーザー管理で社員情報との紐づけを確認してもらってください。

### 登録・編集ができない

権限が不足している可能性があります。

管理者または人事担当者に確認してください。

### 期限や件数が想定と違う

検索条件や絞り込みが残っている可能性があります。

条件をクリアしてから再度確認してください。

## 7. 運用時の注意

- 登録前に入力内容を確認してください。
- 削除操作は慎重に行ってください。
- 期限切れや不適正記録は、確認後に担当者へ連絡してください。
- メール通知は運用対象外です。期限切れや期限間近の確認はダッシュボードと一覧画面で行ってください。
- CSVインポートは、事前チェック結果を確認してから実行してください。
- 権限変更は、利用者が見られる画面に影響します。
